## Folder Structure

- Use `careerfair2020` folder for the angular application
- Use `firebase` folder to backup firebase scripts, rules etc.

## Prerequisites

- Install Angular / Upgrade to Angular 11
  `npm install -g @angular/cli`

## Initial Setup code

```
cd careerfair2020
npm install
```

## Run Application

Inside `careerfair2020` folder
`ng serve`
