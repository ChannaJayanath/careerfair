## Folder Structure

- Use `careerfair2020` folder for the angular application
- Use `firebase` folder to backup firebase scripts, rules etc.

## Prerequisites

- Install Nodejs, npm etc.
- Install Angular / Upgrade to Angular 11
  `npm install -g @angular/cli`

## Initial Setup code

```
cd careerfair2020
npm install
```
